Odoo 19 Module

Clear All Filters Button

One-click remove all active filters, group-bys, and favorites

  The Problem

After applying several filters, group-bys, and favorites in Odoo, clearing them requires clicking the "x" on each individual tag one by one. With complex search combinations this becomes tedious, especially for users who frequently switch between different views and filter sets throughout their workday.

  Key Features

"Clear All" Button in Search Bar

A single button added to the search bar area lets you wipe all active search facets in one click. No more clicking individual "x" icons one at a time.

Removes Filters + Group-Bys + Favorites

Clears everything at once: active filters, group-by groupings, and applied favorites. Returns the view to its clean, unfiltered default state.

Only Appears When Needed

The Clear All button is hidden when no filters are active, keeping the interface clean. It only appears when there are actual facets to remove.

Clean Icon Button

Minimal, unobtrusive design using a clear icon that fits naturally into Odoo's search bar aesthetic. Does not clutter the interface or conflict with existing elements.
