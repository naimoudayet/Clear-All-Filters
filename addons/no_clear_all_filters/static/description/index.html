Clear All Filters

One click to reset every filter, group-by, and favorite

5 clicks become 1. Works on every list and kanban view.

Odoo 19 FREE Zero Config

The Problem

You've applied 5 filters and 2 group-bys. Now you want a clean view. That's 7 clicks on tiny X buttons.

Standard Odoo

Country = France x State = Sale x Salesperson x This Month x

Click X on each one. 4 filters = 4 clicks. Tedious.

With This Module

Country = France State = Sale Clear All

One click. Everything gone. Clean view.
